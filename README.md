# Machine learning project - Comparing models performance

## 🚀 Overview
This is a Python project designed to compare the performance of various machine learning models for predicting house prices based on real estate data. This project aims to provide a comprehensive comparison of different regression models, including Linear Regression, Decision Tree, Random Forest, and Gradient Boosting. By evaluating these models using key metrics such as Mean Absolute Error (MAE) and R² score, developers can gain insights into the strengths and weaknesses of each model.

### Key Features and Benefits
- **Model Comparison**: Easily compare the performance of multiple machine learning models.
- **Data Preprocessing**: Automatically preprocesses the data, including handling datetime and scaling features.
- **Evaluation Metrics**: Provides clear evaluation metrics to assess model performance.
- **Easy to Use**: Simple and straightforward code for quick experimentation.

### Who This Project Is For
- Data scientists and machine learning practitioners
- Students and educators looking to understand model comparison
- Anyone interested in real estate data analysis

## ✨ Features
- 📊 **Data Preprocessing**: Automatically converts date columns and scales features.
- 📈 **Model Training**: Trains multiple regression models.
- 📊 **Evaluation**: Calculates and displays MAE and R² scores for each model.
- 📄 **Data Handling**: Reads and processes CSV data.
